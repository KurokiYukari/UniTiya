Detailed Description

Skill 接口。Skill 是对行为的一种抽象，它会尝试去执行某种行为（但不一定成功）。 Skill 支持两种行为：

TryToPerform 中实现，Cancel 对行为无影响。

TryToPerform 返回 true 时开始，Cancel 时结束。 Cancel 只有在行为开始状态才有实际意义，其余状态对行为无影响。
