Detailed Description

暖暖衣服 Prefab 的组件，用于向外界提供自身的各种信息 衣服绑定到暖暖身上是根据骨骼的名字绑定的 可以有多出的骨骼，请将多出的骨骼新建一个父节点，将父节点置于 AddedBonesRoots 数组中
