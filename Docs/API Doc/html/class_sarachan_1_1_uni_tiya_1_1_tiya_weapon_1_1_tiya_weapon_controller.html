Detailed Description

可配置的武器，通过组合 Skill 构造武器。 ExtraSkills 可以定义对应的 InputAction；InputAction 只有在 Owner.IsPlayer 的情况下才会生效。 所有该武器的 Skill 和 InputAction 的 Enable 只会在改武器 Owner 已被设置的情况下发生，会在设置 Owner 时、组件 Enable 时调用； 而所有 Skil 和 InputAction 的 Disable 会在组件 Disable 时调用。
