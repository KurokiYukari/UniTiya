◆ IsMoving

bool Sarachan.UniTiya.IActorController.IsMoving
get

是否在主动移动。被动的移动（如被击飞、一些动画的 rootMotion 等）都不会使其为 true。 在空中的状态（比如跳跃）会将当前的 IsMoving 状态锁定，直到 IsGround 为 true 才会改变。

◆ ScaledSpeed

float Sarachan.UniTiya.IActorController.ScaledSpeed
get

被缩放到固定的范围的速度，一般用于传递给 Animator 设置动画。

对应于 LocomotionMode ，scaledSpeed 的最大值被限定为： Walk -> 1 ; Run -> 2 ; Sprint -> 3
