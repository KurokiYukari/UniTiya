Detailed Description

可配置的属性同步器。通过反射寻找指定 Unity.Object 中的某个属性，通过 Value 进行同步。

可配置的属性同步器的泛型版本。通过反射寻找指定 Unity.Object 中的某个属性，通过 Value 进行同步。 如果需要序列化，则应该自行通过继承消除泛型并添加 System.SerializableAttribute 特性。
